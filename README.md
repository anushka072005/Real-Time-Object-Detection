# Real-Time Object Detection
A real-time object detection system that identifies and classifies objects from live camera feeds as well as recorded video inputs. The application processes video frames continuously and displays bounding boxes with class labels and confidence values in real time.

## Live Demo
https://youtu.be/hrwAC8bMMO0?si=pzEysHKAnMau2YnQ

## How It Works
1. The system captures input from a live camera feed or video file.
2. Each video frame is processed in real time.
3. Detected objects are highlighted with bounding boxes and labels.
4. The output is displayed continuously on the screen.
